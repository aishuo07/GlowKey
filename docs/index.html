Alpha for macOS

Brightness control that feels built in.

GlowKey keeps MacBook brightness native, then gives external displays clean sliders, cursor-aware shortcuts, and automatic fallback dimming when hardware brightness is blocked.

Install alpha View GitHub
01 No setup

Open it. Drag a slider. Use shortcuts.

02 No jargon

Normal UI never shows DDC, gamma, or VCP.

What makes it useful

One simple layer over messy monitor behavior.

Real when possible

Uses hardware brightness on compatible displays.

Still works otherwise

Falls back to smooth software dimming when adapters block control.

Cursor shortcuts

fn + F1 / F2 changes the external display under your mouse.

Open core

Menu bar app for daily use, CLI for people who script things.
