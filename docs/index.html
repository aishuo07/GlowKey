Alpha for macOS

Your external monitor brightness should feel native.

GlowKey keeps MacBook brightness behavior untouched, then adds simple cursor-aware shortcuts, menu-bar sliders, and automatic fallback dimming for external displays.

What we are building

A small utility that hides the messy display stuff.

01

Works first

Uses real monitor brightness when available, then quietly falls back to software dimming when hardware control is blocked.

02

Mac stays native

Built-in brightness keys keep controlling the MacBook display. GlowKey focuses on external screens instead of fighting macOS.

03

Cursor-aware shortcuts

Use fn + F1 / F2 to change the external display under your mouse. Fallback shortcuts are available when needed.

04

Open-source core

Menu bar app for normal use, CLI for scripting, and a reusable Swift backend for people who want to build on it.

Install

Try the current alpha.

GlowKey is early and currently unsigned. Homebrew is the easiest path; ZIP is available if you prefer manual install.

Homebrew

brew tap aishuo07/glowkey
brew install --cask glowkey
open /Applications/GlowKey.app

If Homebrew asks for tap trust

brew trust --tap aishuo07/glowkey
brew install --cask glowkey

ZIP fallback

Download the latest ZIP from GitHub Releases, move GlowKey.app to Applications, then right-click Open once.
